## 문제 
https://www.acmicpc.net/problem/1629

## 정리
- 재귀 함수
    - 범위를 절반으로 나눈다.
-  모듈러 연산의 특징
    - (a + b) % n = (a % n + b % n) % n;
    - (a - b) % n = (a % n - b % n) % n;
    - (a * b) % n = (a % n * b % n) % n;
- c++일 경우 0을 제외한 숫자는 if 문에서 true 값임.

## 접근 방식
- 재귀로 접근
    - 2 ^ 64 = 2 ^ 32 * 2^32 랑 같다는것을 이용
    - 모듈러 연산을 이용
    - 홀수인 경우의 수를 위해 한 번 더 곱한다.
